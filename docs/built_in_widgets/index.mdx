LVGL ships a set of widgets that can be used directly as XML tags (for
example `<lv_button>`, `<lv_slider>` or `<lv_chart>`). This section
documents the XML API of each one: its properties, parameters and elements,
plus the [enums](./enums) and [style properties](./style_props) that are
shared by every widget.

Every widget builds on the [Base Widget (`lv_obj`)](./lv_obj), so its
properties, flags, events and style bindings are available everywhere. To
learn how the `<api>` itself works — properties, parameters and elements —
see the [API](../syntax/api) page.
