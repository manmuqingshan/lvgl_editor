## Overview

The Spinbox is a numeric editor built on a [Text Area](./lv_textarea). It shows an integer with an optional fixed decimal point and digit count, and the value can be stepped within a range.

Learn more in the [LVGL Open documentation](https://lvgl.io/docs/open/widgets/spinbox).
