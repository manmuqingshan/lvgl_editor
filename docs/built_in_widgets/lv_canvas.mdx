## Overview

The Canvas extends the [Image](./lv_image) widget with a user-provided buffer that LVGL's draw engine can render into. Rectangles, text, images, lines and arcs can all be drawn onto the buffer at runtime.

Learn more in the [LVGL Open documentation](https://lvgl.io/docs/open/widgets/canvas).
