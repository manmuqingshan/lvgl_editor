## Overview

Every widget supports the style properties below. Use them as local
`style_<name>` attributes (optionally with a `-<part>` or `-<state>`
selector) or inside a named [`<style>`](../syntax/styles) sheet. See the
[Styles](../syntax/styles) page for selectors, parts and states.
