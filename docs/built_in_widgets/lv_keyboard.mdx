## Overview

The Keyboard is a specialized [Button Matrix](./lv_buttonmatrix) with predefined keymaps that provides an on-screen virtual keyboard. It can be attached to a [Text Area](./lv_textarea) so key presses edit its text.

Learn more in the [LVGL Open documentation](https://lvgl.io/docs/open/widgets/keyboard).
