## Overview

The Image Button behaves like a [Button](./lv_button) but draws user-supplied images per state instead of a styled rectangle. Separate left, middle and right images can be set; the middle image is tiled to fill the width.

Learn more in the [LVGL Open documentation](https://lvgl.io/docs/open/widgets/imagebutton).
